자율운항선박 경로 계획 연구

2025 - Present

BIT* 기반 경로 계획 알고리즘을 자율운항선박 환경에 적용하고, 해상 장애물 및 목표 선박의 불확실성을 고려한 충돌 회피 경로 생성 성능을 검토한 프로젝트입니다.

프로젝트 개요

자율운항선박의 안전한 항해를 위해 샘플링 기반 경로 계획 기법을 해상 환경에 맞게 적용하고, 목표 선박의 위치 불확실성과 동적 장애물 특성을 반영한 충돌 회피 경로를 생성하는 연구를 수행했습니다.

주요 키워드

  • BIT*
  • Path Planning
  • Collision Avoidance
  • Uncertainty-aware Planning
  • Autonomous Ships

수행 내용

  • 자율운항선박 시나리오에 맞춘 경로 계획 구조 설계
  • 목표 선박 위치 불확실성을 반영한 비용/판단 요소 검토
  • 시뮬레이션 환경에서 경로 생성 및 충돌 회피 성능 확인
  • 국내외 학술대회 및 논문 주제와 연계한 연구 진행

활용 도구

  • Python / C++
  • 시뮬레이션 기반 검증 환경
  • BIT* 기반 경로 계획 알고리즘
