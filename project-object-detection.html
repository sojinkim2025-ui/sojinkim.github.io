PROJECT DETAIL

실선 센서 데이터 기반 해상 객체 탐지

2025 - Present

실제 운항 선박에서 취득된 센서 데이터를 활용하여 해상 객체를 탐지하는 인공지능 모델을 적용하고, 실환경 기반의 성능 검토 및 결과 분석을 수행한 프로젝트입니다.

프로젝트 개요

공개 데이터셋이 아닌 실제 선박 운항 환경에서 취득된 데이터를 활용해 해상 객체(선박, 부유물 등)를 탐지하는 과정을 수행했습니다. 노이즈, 조명 변화, 해상 환경 특성을 고려한 데이터 처리와 결과 해석에 중점을 두었습니다.

주요 키워드

  • Computer Vision
  • Object Detection
  • Sensor Data
  • Maritime AI
  • Real-world Data

수행 내용

  • 실선 환경에서 취득된 센서 데이터 분석
  • 해상 객체 탐지 모델 적용 및 결과 비교
  • 실환경 노이즈 및 장면 특성을 반영한 데이터 정리
  • 탐지 결과 시각화 및 성능 개선 방향 검토

활용 도구

  • Python
  • PyTorch
  • Computer Vision Framework
  • 실선 취득 센서 데이터
